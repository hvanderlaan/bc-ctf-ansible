# bc-ctf-ansible
This repository contains the Ansible code for the BrightCubes Capture The Flag environment.

## Usage
```
git clone git@github.com/hvanderlaan/bc-ctf-ansible.git
cd bc-ctf-ansible
ansible-playbook site.yaml --ask-vault-pass
```
